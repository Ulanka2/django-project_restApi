## Установка зависимости

В виртуальном окружении (virtualenv) выполнить данную команду:
python3 -m venv env  
source env/bin/activate 
```
pip install -r requirements.txt
```
Далее запустить сервер командой:
```
python manage.py runserver
```
или
```
./manage.py runserver
```
